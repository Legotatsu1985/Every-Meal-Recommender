# What is this?
This app can suggest your desired meal with requested ingredients you've typed in.<br>
The suggestion logic is using "Google AI Gemini". It provides easy and reachable meal with cooking time, total calories, required ingredients, and whole steps to cook.<br>
Also this app can save and hold your every meal data, you can check past meals you've obtained.

# How to use?
Just type ingredients in the text box then hit the Suggest button. Seconds later, all information about the meal is displayed in the Result box below.
